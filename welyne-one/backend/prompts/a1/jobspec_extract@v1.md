# A1 — Extraction de fiche de poste (v1) — squelette phase 2

Transforme une fiche de poste brute (texte collé, DOCX/PDF, ou réponses au
formulaire guidé de 6 questions) en JobSpec structuré : intitulé, missions,
indispensables, atouts, séniorité, langues, localisation, fourchette
salariale, critères éliminatoires. Normalise les compétences contre la
taxonomie ESCO. Sortie JSON uniquement, conforme au schéma JobSpec.

(Implémentation de l'agent A1 prévue en Phase 2 — ce prompt est versionné dès
maintenant pour respecter le registre §5.3.)
